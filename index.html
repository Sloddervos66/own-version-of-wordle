How it works

This works like wordle, but every letter combination is possible, instead of it being limited to the dictionary!


Try guessing a 5-character word.
If the background is gray, then it's incorrect.
If it's this color, then it's a correct word in the wrong place
If it's this color, it's the correct word in the correct place.
If you cannot tell the difference between these colors, toggle high contrast mode.
